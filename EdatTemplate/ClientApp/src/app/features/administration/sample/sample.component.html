Sample Data Component
This component demonstrates the use of a store service to manage client-side application state. The data grid supports sorting, filtering, and paging, and is composed of smaller components that implement these features. Adding, updating, and deleting data is implemented via modal dialogs, and all validations are performed server-side. This component also demonstrates exporting the filtered data to Excel, and using moment.js to calculate the age based on the birthdate client-side.

Sample Email Component
This component demonstrates the use of the email service to send an email to a selected user using the SendGrid API. The user is selected with the Staff Picker component that uses the Azure API Management FDOT core services server-side.

Sample Documents Component
This component demonstrates the use of the BLOB storage service to store documents and files. The BLOB storage service uses Azure BLOB Storage server-side.
